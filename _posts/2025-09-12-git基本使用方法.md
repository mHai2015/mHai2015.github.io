# Git 基本使用方法

本文介紹 Git 的一些常用操作，幫助初學者快速上手。

---

## 1. Git 配置

安裝 Git 後，首先需要設置用戶名與郵箱：

```bash
git config --global user.name "你的名字"
git config --global user.email "你的郵箱"
```
列出git設置
要驗證本地倉庫的git設置，請使用以下命令：
```bash
git config --list
```

## 1. Git 分支管理策略
創建並切換到dev分支

```bash
git checkout -b dev
```
```git checkout``` 命令加上 ```-b``` 參數表示創建並切換，相當於以下兩條命令：
```bash
git branch dev
git checkout dev
```
查看當前分支：
```bash
git branch
```
將新分支dev推送到遠程端
```bash
git push origin dev
```
查看遠程默認分支：
```bash
git remote show origin
```
修改遠程默認分支：
```bash
git remote set-head origin dev
```
